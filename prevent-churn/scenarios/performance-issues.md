# Scenario: Performance/Speed Issues

## Common Issues
- Slow loading or processing times
- Timeout errors during operations
- Server resource limits being hit
- Backup or migration processes hanging

## Key Solutions to Offer
- Performance audit of their site and server setup
- Server optimization recommendations
- Upgrade path with a trial period
- Direct server configuration assistance

## Example Opening
"I'm sorry you're experiencing slowdowns — there's nothing more frustrating than watching a process hang or time out when you just need to get work done. I'd really like to help you get the performance you deserve."

## Approach Pattern
1. **Identify the bottleneck** — server resources, database size, hosting tier
2. **Audit first** — offer to review their setup before recommending changes
3. **Specific recommendations** — not vague advice, but exact settings to adjust
4. **Upgrade path** — if their current hosting can't support the operation, show them a clear path forward
5. **Assisted setup** — offer to configure the changes yourself
6. **Safety net** — refund if performance can't be improved to their satisfaction

## Why This Approach Works
- Offering an audit before prescribing solutions shows thoroughness, not guesswork
- Specific recommendations build confidence that you know what you're doing
- Showing an upgrade path respects their budget while giving them a clear option
