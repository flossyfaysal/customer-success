# Scenario B: Integration & Compatibility Questions

**When to use:** Customer asks about compatibility with specific platforms, hosting providers, cloud storage services (OneDrive, Google Drive, Dropbox, Amazon S3, etc.), or WordPress configurations.

## Approach
- Confirm which integrations are supported
- Be specific about which plan supports which integrations (Basic = 1 remote storage, Plus+ = unlimited)
- For cross-server restore from third-party cloud: be upfront about the limitation, then cross-sell Duplicator Cloud Storage
- Include setup guide links for relevant integrations

## Key Limitation to Always Communicate
Cross-server migration from 3rd-party cloud storage (OneDrive, Google Drive, etc.) is NOT supported through the recovery interface. Use Duplicator Cloud Storage for that.
