# Tone Calibration

## By Follow-Up Number

### First Follow-Up (3-5 days after last contact)
- **Tone**: Casual, warm, curious
- **Length**: Short (6-8 sentences)
- **Energy**: "Just checking in!"

### Second Follow-Up (7-10 days, no response)
- **Tone**: Still warm, slightly more direct about offering help
- **Length**: Standard (8-10 sentences)
- **Energy**: "I'm still here and want to make sure you're not stuck"

### Final Follow-Up (14+ days, no response)
- **Tone**: Warm but clear this is a last check-in
- **Length**: Concise (6-8 sentences)
- **Energy**: "The door is always open"
- **Note**: Can combine with Prompt Library > Final Follow-Up template

## Quick-Start Checklist

When crafting your CX Approach reply, check:

- [ ] Did I reference the previous conversation or issue specifically?
- [ ] Did I ask about the current status warmly (not demandingly)?
- [ ] Did I use their site activation info (if available) to personalize?
- [ ] Did I offer hands-on help (site access, done-for-you)?
- [ ] Did I reinforce the continuous success goal?
- [ ] Did I include a service reminder only if it fits naturally?
- [ ] Did I keep the tone warm, human, and non-pushy?
- [ ] Did I make it easy for them to respond?
- [ ] If the customer shared personal life context, did I lead with that genuinely?
- [ ] If this is a closure/refund, did I process it with zero friction and confirm details?
- [ ] If the customer is leaving, did I leave the door open warmly for a future return?
- [ ] If they purchased but haven't activated, did I mention it gently without pressure?
- [ ] If there's an unresolved issue with no feedback, did I acknowledge it without guilt-tripping?
- [ ] Did I offer to do the actual work (not just investigate) when appropriate?
- [ ] Is this paste-ready with no commentary or metadata?
