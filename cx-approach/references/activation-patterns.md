# Activation Status Patterns

When you have site activation data, use these patterns to personalize the message:

| Status | What to Say |
|---|---|
| Active on multiple sites | "Great to see you're using Duplicator on [sites] — hopefully backups are running smoothly!" |
| Activated then deactivated | "I noticed [sites] were activated but are currently deactivated — I just want to make sure nothing went wrong" |
| Purchased but never activated | "I noticed you've purchased your license but haven't activated it on any site yet — no rush, but I want to make sure nothing is blocking you" |
| Never activated (no purchase context) | "I see you haven't had a chance to activate yet — no rush, but I'm here to help whenever you're ready" |
| Active on some, deactivated on others | "I see [active site] is running — great! I noticed [deactivated site] isn't active right now. Want me to take a look?" |
